[[abstract]]
== Abstract

The vertebrates template provides standardized metadata fields for non-human vertebrate proteomics experiments. This includes mammals (mouse, rat, etc.), birds, fish, reptiles, and amphibians.

This template is appropriate for:

* Mouse and rat proteomics studies
* Model organism research (zebrafish, Xenopus)
* Livestock and agricultural animal studies
* Wildlife and comparative proteomics

[[extends]]
== Template Hierarchy

[source]
----
base (construction artifact)
  └── vertebrates (SAMPLE layer - must combine with TECHNOLOGY template)
----

The vertebrates template extends the **base** template, adding disease annotation and vertebrate-specific fields.

[IMPORTANT]
====
The vertebrates template **cannot be used alone**. You must combine it with a technology template:

* `ms-proteomics` + `vertebrates` - For MS-based proteomics of vertebrate samples
* `affinity-proteomics` + `vertebrates` - For affinity proteomics of vertebrate samples
====

[[checklist]]
== Checklist

=== Inherited Columns

All columns from the base template are inherited (source name, organism, organism part, biological replicate, assay name, technology type, instrument, technical replicate, data file). Technology-specific columns come from the technology template (`ms-proteomics` or `affinity-proteomics`).

[[developmental-stages]]
== Developmental Stages

Common developmental stage terms for vertebrates:

|===
|Stage |EFO/UBERON Term |Description

|adult |EFO:0001272 |Sexually mature organism
|embryo |UBERON:0000068 |Organism in early development
|juvenile |EFO:0001303 |Post-embryonic, pre-adult
|neonate |EFO:0001375 |Newborn
|fetus |UBERON:0000323 |Late embryonic stage (mammals)
|===
