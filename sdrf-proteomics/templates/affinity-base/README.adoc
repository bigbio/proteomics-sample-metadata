[[abstract]]
== Abstract

The Affinity-Base template is the **minimum valid SDRF template** for affinity-based proteomics experiments. It includes all columns necessary to describe an affinity proteomics dataset.

This template is appropriate for:

* Olink proteomics (PEA technology)
* SomaScan proteomics (aptamer-based)
* Luminex assays (bead-based)
* RPPA (Reverse Phase Protein Array)
* ELISA-based proteomics
* Other antibody or aptamer-based protein detection

[[hierarchy]]
== Template Hierarchy

[source]
----
base
  └── affinity-base (THIS TEMPLATE)
        │
        └── affinity-proteomics (additional affinity-specific columns)
----

=== Columns NOT in This Template

[NOTE]
====
The following MS-specific columns are **NOT** included in affinity templates:

* `comment[proteomics data acquisition method]` - No MS acquisition
* `comment[cleavage agent details]` - No protein digestion
* `comment[label]` - No isotopic labeling
* `comment[fraction identifier]` - No fractionation
====

[[usage]]
== Usage

=== Minimal Affinity Proteomics

[source]
----
#template=affinity-base; version=v1.1.0
----

=== With Sample Metadata (Recommended)

[source]
----
#template=affinity-base; version=v1.1.0
#template=human; version=v1.1.0
----

=== With Additional Affinity Parameters

[source]
----
#template=affinity-base; version=v1.1.0
#template=human; version=v1.1.0
#template=affinity-proteomics; version=v1.0.0
----

[[validation]]
== Validation

[source,bash]
----
pip install sdrf-pipelines
parse_sdrf validate-sdrf --sdrf_file your_file.sdrf.tsv --template affinity-base
----
