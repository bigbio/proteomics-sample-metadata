[[abstract]]
== Abstract

The plants template provides standardized metadata fields for plant proteomics experiments. This includes model organisms like Arabidopsis thaliana, as well as crop species, trees, and other plant systems.

This template is appropriate for:

* Arabidopsis proteomics studies
* Crop proteomics (rice, maize, wheat, soybean)
* Plant stress response studies
* Agricultural and plant biology research

[[extends]]
== Template Hierarchy

[source]
----
base (construction artifact)
  └── plants (SAMPLE layer - must combine with TECHNOLOGY template)
----

The plants template extends the **base** template, adding disease annotation and plant-specific fields.

[IMPORTANT]
====
The plants template **cannot be used alone**. You must combine it with a technology template:

* `ms-proteomics` + `plants` - For MS-based proteomics of plant samples
* `affinity-proteomics` + `plants` - For affinity proteomics of plant samples
====

[[checklist]]
== Checklist

=== Inherited Columns

All columns from the base template are inherited (source name, organism, organism part, biological replicate, assay name, technology type, instrument, technical replicate, data file). Technology-specific columns come from the technology template (`ms-proteomics` or `affinity-proteomics`).

[[developmental-stages]]
== Developmental Stages

Common developmental stages from the Plant Ontology (PO):

|===
|Stage |PO Term |Description

|seedling stage |PO:0007131 |Plant after germination before mature vegetative growth
|rosette growth stage |PO:0007113 |Arabidopsis-specific rosette development
|vegetative growth stage |PO:0007134 |Mature vegetative plant
|reproductive growth stage |PO:0007130 |Flowering and fruiting
|flowering stage |PO:0007616 |Active flowering
|fruit formation stage |PO:0007042 |Fruit/seed development
|senescent stage |PO:0025337 |Aging and senescence
|===

|seed |PO:0009010 |Mature ovule
|fruit |PO:0009001 |Mature ovary
|shoot |PO:0009006 |Aerial part

|===

[[growth-conditions]]
== Growth Conditions

Document growth conditions using standard descriptions:

|===
|Condition |Format |Example

|Photoperiod |{light}h light/{dark}h dark |16h light/8h dark
|Temperature |{temp}°C |22°C
|Light intensity |{value} μmol m⁻² s⁻¹ |120 μmol m⁻² s⁻¹
|Growth medium |{medium name} |1/2 MS medium
|===

[[treatments]]
== Common Plant Treatments

|===
|Treatment Type |Example Values |Ontology

|Abiotic stress |drought stress, salt stress, cold stress, heat stress |EFO
|Biotic stress |pathogen infection, herbivore attack |EFO
|Hormone treatment |auxin treatment, ABA treatment, jasmonic acid |ChEBI
|Nutrient stress |nitrogen starvation, phosphate starvation |EFO
|===
