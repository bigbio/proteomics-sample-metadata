[[status]]
== Status of this Template

This document provides a template for annotating crosslinking mass spectrometry (XL-MS) experiments in SDRF-Proteomics format. This template defines additional columns that can be combined with any core template (human, vertebrates, etc.) to capture crosslinking-specific metadata.

**Type:** Specialized Template (used in combination with core templates)

**Status:** Draft - This template is under active development and may change significantly.

**Version:** 1.0.0-dev - 2025-01

[[abstract]]
== Abstract

Crosslinking mass spectrometry (XL-MS) uses chemical reagents to covalently link amino acid residues in close spatial proximity, providing information for protein structure determination, protein-protein interaction mapping, complex characterization, and integrative structural biology.

XL-MS experiments have unique characteristics: crosslinking chemistry with different specificities and spacer arm lengths, distance constraints, reaction conditions affecting efficiency, specialized search engines for crosslinked peptide pairs, and enrichment strategies (SEC, SCX, affinity). This template defines the additional metadata requirements for annotating XL-MS datasets.

[[connections]]
== Connections to Structural Biology

Crosslinking mass spectrometry data is typically integrated with:

- **Cryo-EM structures**: As distance restraints for modeling
- **X-ray crystallography**: For structure validation
- **AlphaFold predictions**: For model assessment
- **Integrative modeling platforms**: IMP, HADDOCK, Rosetta
- **Protein Data Bank (PDB)**: For structure deposition

When structures are available, linking to PDB identifiers is RECOMMENDED.

[[additional-ontologies]]
== Additional Ontologies

In addition to the ontologies supported by the core SDRF-Proteomics specification, crosslinking templates utilize:

- **PSI-MS CV**: Contains crosslinker-specific terms (MS:1002494 - crosslink donor, etc.)
- **ChEBI**: For chemical compound identification
- **XLMOD**: Crosslinking modification database (https://www.ebi.ac.uk/ols4/ontologies/xlmod)

[[checklist]]
== Checklist

This section defines the metadata columns required and recommended for crosslinking experiments.

=== Sample-Level Columns (characteristics)

These columns describe the sample itself and allow identification of crosslinking samples when querying sample metadata:

|===
|Column Name |Description |Cardinality |Ontology/CV |Example Values

|characteristics[crosslinking reagent]
|Name of the cross-linking reagent used (identifies this as a crosslinking sample)
|1
|XLMOD, ChEBI
|DSSO, DSS, BS3, EDC, DSBU

|characteristics[enrichment process]
|Indicates if the sample was enriched for cross-linked peptides
|1
|Controlled vocabulary
|enrichment of cross-linked peptides, not applicable

|characteristics[crosslink type]
|Type of crosslinks being studied
|1
|Controlled vocabulary
|homo-bifunctional, hetero-bifunctional, zero-length, photo-reactive, cleavable
|===

=== Data-Level Columns (comment)

These columns contain technical details about the cross-linking experiment:

|===
|Column Name |Description |Cardinality |Ontology/CV |Example Values

|comment[cross-linking method]
|MS-based cross-linking methodology used
|1
|Controlled vocabulary
|cross-linking mass spectrometry, hydrogen-deuterium exchange, ion-mobility mass spectrometry

|comment[cross-linker]
|Cross-linker compound with structured properties (see <<crosslinker-annotation>>)
|1
|XLMOD, ChEBI, PRIDE
|NT=DSSO;AC=XLMOD:02126;TA=K;CL=yes;MH=54.01;ML=85.05
|===

NOTE: The `characteristics[crosslinking reagent]` provides a simple identifier for sample-level queries, while `comment[cross-linker]` contains the detailed structured annotation with all cross-linker properties.

=== Recommended Columns

The following columns are RECOMMENDED for crosslinking experiments:

|===
|Column Name |Description |Cardinality |Ontology/CV |Example Values

|characteristics[crosslink distance]
|Maximum Ca-Ca distance constraint provided by the crosslinker
|0..1
|Numeric value with unit (Angstroms)
|11.4 A, 24.0 A, 30.0 A, 26.4 A

|characteristics[crosslinker spacer arm length]
|Length of the crosslinker spacer arm
|0..1
|Numeric value with unit (Angstroms)
|11.4 A, 7.7 A, 12.0 A

|characteristics[crosslinking reaction time]
|Duration of the crosslinking reaction
|0..1
|Numeric value with unit
|30 min, 60 min, 2 h

|characteristics[crosslinking temperature]
|Temperature at which crosslinking was performed
|0..1
|Numeric value with unit (Celsius)
|25 C, 37 C, 4 C, room temperature

|comment[crosslinker concentration]
|Concentration of crosslinking reagent used
|0..1
|Numeric value with unit
|2 mM, 500 uM, 1 mM, 1:50 protein ratio

|comment[crosslinker to protein ratio]
|Molar ratio of crosslinker to protein
|0..1
|Ratio format
|50:1, 100:1, 1:1 w/w

|comment[quenching reagent]
|Reagent used to quench the crosslinking reaction
|0..1
|Chemical name
|Tris-HCl, ammonium bicarbonate, glycine, hydroxylamine

|comment[crosslink enrichment method]
|Method used to enrich crosslinked peptides
|0..1
|Controlled vocabulary or free text
|size exclusion chromatography, strong cation exchange, affinity enrichment, peptide SEC

|comment[crosslink target residues]
|Amino acid residues targeted by the crosslinker
|0..1
|Amino acid codes
|K-K, K-S-T-Y, D-E to K, C-C
|===

=== Key-Value Annotation Format

For detailed crosslinker annotation, use the key-value format in the `comment[cross-linker]` column:

[source]
----
NT=DSSO;AC=XLMOD:02126;TA=K;CL=yes;MH=54.01;ML=85.05
----

The following properties are supported:

|===
|Property |Key |Description |Status |Example

|Cross-linker Name
|NT
|Name of the cross-linking reagent
|REQUIRED
|NT=DSSO

|Cross-linker Accession
|AC
|Ontology accession (XLMOD, ChEBI, or PRIDE)
|REQUIRED
|AC=XLMOD:02126

|Target Amino Acid
|TA
|Target residue(s) for the cross-linker
|RECOMMENDED
|TA=K

|Cleavable
|CL
|Whether the cross-linker is MS-cleavable
|RECOMMENDED
|CL=yes, CL=no

|Mass Heavy Fragment
|MH
|Mass of heavy fragment for cleavable cross-linkers (Da)
|RECOMMENDED (if cleavable)
|MH=54.01

|Mass Light Fragment
|ML
|Mass of light fragment for cleavable cross-linkers (Da)
|RECOMMENDED (if cleavable)
|ML=85.05

|Hydrolyzed Mass
|HM
|Hydrolyzed mass for non-cleavable cross-linkers (Da)
|RECOMMENDED (if non-cleavable)
|HM=156.08

|Spacer Arm Length
|SA
|Spacer arm length in Angstroms
|RECOMMENDED
|SA=10.3
|===

.Examples
[source]
----
# MS-cleavable cross-linker (DSSO)
NT=DSSO;AC=XLMOD:02126;TA=K;CL=yes;MH=54.01;ML=85.05;SA=10.3

# Non-cleavable cross-linker (DSS)
NT=DSS;AC=XLMOD:02001;TA=K;CL=no;HM=156.08;SA=11.4

# Zero-length cross-linker (EDC)
NT=EDC;AC=XLMOD:02009;TA=D,E-K;CL=no;SA=0
----

|CID, HCD, ETD

Crosslinked peptides are typically low abundance. Document enrichment approaches:

|===
|Enrichment Method |Description |Typical Use Case

|Size exclusion chromatography (SEC)
|Separates crosslinked peptides by size
|General XL-MS workflows

|Strong cation exchange (SCX)
|Crosslinked peptides have higher charge states
|General XL-MS workflows

|Affinity enrichment
|Uses specific tags on cleavable crosslinkers
|PIR, azide-alkyne click chemistry

|High-pH reversed phase
|Orthogonal separation to standard RP
|Fractionation strategies

|FAIMS/ion mobility
|Gas-phase separation of crosslinked peptides
|Advanced instrument methods
|===

[[best-practices]]
== Best Practices for Crosslinking Annotation

1. **Always specify crosslinker**: Document the exact crosslinking reagent used, including manufacturer if relevant.

2. **Report distance constraints**: Include spacer arm length and maximum Ca-Ca distance for structural interpretation.

3. **Document reaction conditions**: Time, temperature, and concentration affect crosslinking efficiency and reproducibility.

4. **Specify enrichment method**: Crosslink enrichment is critical for data quality and should be documented.

5. **Note cleavability**: For MS-cleavable crosslinkers, indicate this property as it affects data analysis strategy.

6. **Use controlled vocabularies**: Prefer XLMOD or PSI-MS CV terms for crosslinker identification.

7. **Include quenching information**: Document how the reaction was stopped.

8. **Link to structures**: When available, reference related PDB entries for structural context.
