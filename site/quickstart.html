Create Your First SDRF File in 5 Minutes

SDRF-Proteomics is a simple tab-separated file that describes your proteomics samples and links them to your data files.

Step 1: Choose Your Template

Answer this question to find your template:

Your Sample Type Template to Use Download
Human samples human Download
Mouse, rat, zebrafish vertebrates Download
Insects, worms invertebrates Download
Plants plants Download
Cell lines cell-lines Download
Other / Not sure default Download

Step 2: Fill in Your Sample Information

Open your template in Excel or any spreadsheet software. For each sample, fill in:

Column What to Write Example
source name A unique name for your sample patient_001
characteristics[organism] Species name (lowercase) homo sapiens
characteristics[organism part] Tissue or body part liver
characteristics[disease] Disease name, or "normal" if healthy hepatocellular carcinoma

Step 3: Fill in Your Data File Information

For each row, also fill in information about the raw file:

Column What to Write Example
assay name A name for this MS run run_001
comment[label] Type of labeling label free sample
comment[instrument] Mass spectrometer used Q Exactive HF
comment[data file] Your raw file name sample_001.raw

Step 4: Add Your Experimental Variables

If you're comparing groups (e.g., disease vs. healthy), add a factor value column:

Column What to Write Example
factor value[disease] The variable you're studying normal or hepatocellular carcinoma

Step 5: Validate Your File

Save your file as .sdrf.tsv and validate it:

pip install sdrf-pipelines
parse_sdrf validate-sdrf --sdrf_file your_file.sdrf.tsv

Complete Example

Here's a complete SDRF file for a simple human liver cancer study:

source name characteristics[organism] characteristics[organism part] characteristics[disease] assay name comment[label] comment[instrument] comment[data file] factor value[disease]
patient_001 homo sapiens liver hepatocellular carcinoma run_001 label free sample Q Exactive HF patient_001.raw hepatocellular carcinoma
patient_002 homo sapiens liver hepatocellular carcinoma run_002 label free sample Q Exactive HF patient_002.raw hepatocellular carcinoma
control_001 homo sapiens liver normal run_003 label free sample Q Exactive HF control_001.raw normal
control_002 homo sapiens liver normal run_004 label free sample Q Exactive HF control_002.raw normal
Sample metadata Data file metadata Factor values

Common Mistakes to Avoid

Mistake Correct
Source Name (capitalized) source name (lowercase)
characteristics [organism] (space before bracket) characteristics[organism] (no space)
control for healthy samples normal for healthy samples
Leaving cells empty Use not available or not applicable
sourcename (no space) source name (with space)

Where to Find Correct Values

For This Field Look Here
Organism names NCBI Taxonomy
Tissue/organ names UBERON
Disease names MONDO
Cell types Cell Ontology
Instruments PSI-MS
Cell lines Cellosaurus

Tip: Don't worry about finding the exact ontology term. Just write the common name (e.g., "liver", "breast cancer") and the validator will check it for you.
