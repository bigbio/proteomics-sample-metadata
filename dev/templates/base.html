1. Status

Status: Internal Construction Artifact

Version: 1.1.0 - 2026-01

Important

This template cannot be used directly. It is an internal construction artifact that defines shared columns inherited by all other templates.

For documentation of base columns, see the main SDRF-Proteomics specification, specifically the sample metadata and data file metadata sections.

2. Usage

Use one of the following technology templates instead:

  • MS-Proteomics Template - For mass spectrometry-based proteomics

  • Affinity-Proteomics Template - For affinity-based proteomics (Olink, SomaScan)

3. Template Files

  • base.yaml - YAML template definition for validation

  • base-template.sdrf.tsv - Column header template
