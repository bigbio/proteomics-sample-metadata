1. Status

Status: Released

Version: 1.1.0 - 2026-01

2. Abstract

The Affinity-Base template is the minimum valid SDRF template for affinity-based proteomics experiments. It includes all columns necessary to describe an affinity proteomics dataset.

This template is appropriate for:

  • Olink proteomics (PEA technology)

  • SomaScan proteomics (aptamer-based)

  • Luminex assays (bead-based)

  • RPPA (Reverse Phase Protein Array)

  • ELISA-based proteomics

  • Other antibody or aptamer-based protein detection

3. Template Hierarchy

base
  └── affinity-base (THIS TEMPLATE)

        └── affinity-proteomics (additional affinity-specific columns)

4. Checklist

4.1. Inherited Columns (from base)

ColumnRequirementDescription
source name REQUIRED Unique identifier for the biological sample
characteristics[organism] REQUIRED Species (NCBI Taxonomy)
characteristics[organism part] REQUIRED Anatomical part
characteristics[cell type] RECOMMENDED Cell type
characteristics[biological replicate] REQUIRED Biological replicate number
assay name REQUIRED Unique identifier for the assay run
technology type REQUIRED "protein expression profiling by antibody array" or "protein expression profiling by aptamer array"
comment[technical replicate] RECOMMENDED Technical replicate number
comment[data file] REQUIRED Data file name

4.2. Affinity-Specific Columns

ColumnRequirementDescriptionExample Values
comment[instrument] REQUIRED Affinity platform/instrument used Olink Explore HT, SomaScan Assay 7K, Luminex 200
characteristics[affinity method] REQUIRED Affinity proteomics platform Olink, SomaScan, Luminex, RPPA, ELISA
characteristics[affinity reagent] REQUIRED Type of affinity reagent antibody, aptamer, nanobody, affimer
comment[panel name] RECOMMENDED Commercial panel name Olink Explore 3072, SomaScan 7K, Luminex Human Cytokine Panel
comment[quantification unit] RECOMMENDED Unit of quantification NPX, RFU, MFI, pg/mL

4.3. Columns NOT in This Template

Note

The following MS-specific columns are NOT included in affinity templates:

  • comment[proteomics data acquisition method] - No MS acquisition

  • comment[cleavage agent details] - No protein digestion

  • comment[label] - No isotopic labeling

  • comment[fraction identifier] - No fractionation

5. Usage

5.1. Minimal Affinity Proteomics

#template=affinity-base; version=v1.1.0

5.2. With Sample Metadata (Recommended)

#template=affinity-base; version=v1.1.0
#template=human; version=v1.1.0

5.3. With Additional Affinity Parameters

#template=affinity-base; version=v1.1.0
#template=human; version=v1.1.0
#template=affinity-proteomics; version=v1.0.0

6. Validation

pip install sdrf-pipelines
parse_sdrf validate-sdrf --sdrf_file your_file.sdrf.tsv --template affinity-base
