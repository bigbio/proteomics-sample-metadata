2. Abstract

The invertebrates template provides standardized metadata fields for invertebrate proteomics experiments. This includes model organisms like Drosophila melanogaster and Caenorhabditis elegans, as well as other insects, worms, and invertebrate species.

This template is appropriate for:

  • Drosophila (fruit fly) proteomics studies

  • C. elegans proteomics studies

  • Insect proteomics (bees, mosquitoes, etc.)

  • Marine invertebrate studies

3. Template Hierarchy

base (construction artifact)
  └── invertebrates (SAMPLE layer - must combine with TECHNOLOGY template)

The invertebrates template extends the base template, adding disease annotation and invertebrate-specific fields.

Important

The invertebrates template cannot be used alone. You must combine it with a technology template:

  • ms-base + invertebrates - For MS-based proteomics of invertebrate samples

  • affinity-proteomics + invertebrates - For affinity proteomics of invertebrate samples

4. Checklist

4.1. Inherited Columns

All columns from the base template are inherited (source name, organism, organism part, biological replicate, assay name, technology type, instrument, technical replicate, data file). Technology-specific columns come from the technology template (ms-base or affinity-proteomics).

4.2. Invertebrates-Specific Columns

ColumnRequirementDescriptionOntologyExample Values
characteristics[disease] REQUIRED Disease state of the sample MONDO, EFO, DOID, PATO normal, breast cancer, diabetes mellitus
characteristics[developmental stage] RECOMMENDED Developmental stage of the organism FBdv (Drosophila), WBls (C. elegans), EFO, UBERON adult stage, larval stage, pupal stage, embryonic stage
characteristics[strain/breed] OPTIONAL Strain of the organism NCBITaxon Oregon-R, w1118, N2, Canton-S
characteristics[genotype] OPTIONAL Genotype of the organism — wild type, daf-2(e1370), w[*]; P{GAL4}

7. Genotype Notation

7.1. Drosophila Genotype Format

Drosophila genotypes follow a specific notation:

# Wild type
wild type

# Single mutation
w[1118]

# Multiple alleles with GAL4/UAS system
w[*]; P{GAL4-da.G32}

7.2. C. elegans Genotype Format

  1. elegans genotypes use allele notation:

# Wild type
wild type

# Single mutation (gene-allele format)
daf-2(e1370)

# Multiple mutations
daf-2(e1370); daf-16(mgDf50)

8. Validation

Validate SDRF files using sdrf-pipelines:

pip install sdrf-pipelines
